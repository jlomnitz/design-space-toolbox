Detailed Description

Data type representing a GMA-System.

This data structure is a standard representation of an GMA using matrix notation. Here, the positive and negative terms are explicitly represented according to the Gs and Hs. Also, matrices are split up relating to either dependent and independent parameters. The GMA system uses an array of matrices to represent all the terms in all of the equations.
