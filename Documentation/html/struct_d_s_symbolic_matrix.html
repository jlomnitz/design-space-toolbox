Detailed Description

Data type representing a symbolic matrix.

This data type is the front end of the matric manipulation portion of the design space toolbox involving symbolic data.. Currently, the DST library has a very limited manipulation of symbolic libraries, and is used exclusive to parse gma equations and design spaces. When performing any analysis of design space, the symbolic matrices are converted to numerical expressions.
