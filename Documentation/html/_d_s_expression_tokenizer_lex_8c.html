Detailed Description

Implementation file with functions for tokenizing matrices, generated by flex.

This file was generated directly by the flex program, and is the source code responsible for matrix tokenization. This file was generated by flex, according to a specification written by Jason Lomnitz. To generate this file, the following command must be executed: "flex -t DSExpressionGrammar.l > DSExpressionTokenizerLex.c".
